Formulário de Feedback - Curso de Programação Web

Preencha o formulário abaixo para avaliarmos sua experiência no curso.

Campo obrigatório: Este campo usa o atributo required para garantir que seja preenchido.

Campo obrigatório: O navegador verifica se o formato digitado é um e-mail ou um número válido.

0 atributo min define o valor mínimo e max o valor máximo permitido.

O tipo date exibe um seletor de calendário para facilitar a escolha da data.

Usamos a tag <select> para criar um menu suspenso com opções pré-definidas.

A tag <textarea> permite escrever textos mais longos.

0 required garante que o usuário só possa enviar se marcar essa opção.

O botão <button type="submit"> envia os dados do formulário.
